# P3-Assembly-Sublime-Text-Syntax
Syntax for P3 Assembly in Sublime Text 3,2

###V0.01

## Install
On Sublime text: 
- Preferences -> Browse Packages
- Go up one folder
- Go into Installed Packages
- Paste *.sublime-package there
- Restart Sublime Text
